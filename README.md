# Permission Viewer by AP

**Permission Viewer by AP** is a lightweight and powerful Android utility designed to help users and developers explore the inner workings of their installed applications and standalone APK files. Built with a modern **Material 3 (Material You)** interface, it provides a clean and intuitive way to inspect permissions, services, and technical metadata.

## 🚀 Features

- **Installed Apps List**: Quickly browse through all applications installed on your device.
- **APK Inspector**: Load and extract detailed information from any APK file stored on your device.
- **Search & Filter**: Find specific apps instantly using the real-time search functionality.
- **Detailed Information**:
    - **Permissions**: Complete list of requested permissions.
    - **Components**: View all Activities, Services, Broadcast Receivers, and Content Providers.
    - **Metadata**: Package name, Version (Name & Code), SDK info (Min/Target SDK), UID, and Install Times.
    - **Security**: View App Signatures (SHA-256 thumbprints).
- **Customizable Appearance**:
    - **Theme Support**: Switch between Light, Dark, and System Default modes.
    - **AMOLED Mode**: True black background for OLED screens.
    - **Dynamic Colors**: Support for Android 12+ Material You dynamic color palettes.
- **Easy Sharing**: One-tap "Copy All" feature to save app details to your clipboard.

## 🛠 Tech Stack

- **Language**: Kotlin
- **UI Framework**: XML with Material Components (Material 3)
- **Architecture**: Modern Android Best Practices
- **Dependencies**:
    - `androidx.appcompat`: For backward compatibility.

## 📦 Building the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/ap0803apap-sketch/PermissionViewerbyAP.git
   ```
2. Open the project in **Android Studio (Ladybug or newer)**.
3. Sync the project with Gradle files.
4. Build and Run on an emulator or a physical device (Android 7.1+ / API 25+).
